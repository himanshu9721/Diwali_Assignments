1. You are given an array of strings. Your task is to determine the length of the
longest string in the array?

public class LongestStringLength {
    public static void main(String[] args) {
        // Example array of strings
        String[] arr = {"apple", "banana", "kiwi", "strawberry", "grapes"};

        int maxLength = 0;

        for (String str : arr) {
            if (str.length() > maxLength) {
                maxLength = str.length();
            }
        }

        System.out.println("The length of the longest string is: " + maxLength);
    }
}


output:
The length of the longest string is: 10
